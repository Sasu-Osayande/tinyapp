## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- body-parser
- cookie-session
- morgan

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` or the `npm start` command.
